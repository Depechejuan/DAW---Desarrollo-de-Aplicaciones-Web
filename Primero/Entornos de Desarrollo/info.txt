Paradigmas de programación.
BASIC - bueno para comenzar a programar.
COBOL - Utilizado en banca, en desuso en el resto de sitios.
FORTRAN - Concebido para ingenieria y aplicaciones uqe requieren uso exhaustivo de operaciones matematicas.
ENSAMBLADOR - Bajo nivel, cercano a máquina. Sustituye 0 y 1 por palabras más fáciles.
C - Considerado de los mejores junto a C++ y Java. No muy complicado de parender y con gran control.
C++ - A partir de C, orientado a objetos.
JAVA - A partir de C++, elimina algunos de sus inconvenientes y tiene gran difusión.
PHP - Interpretado, Diseñado para webs dinámicas. Del lado del servidor
PYTHON - Alto nivel, cuya filosofía hace hincapié ne una sintaxis muy limpia y que favorece código legible. Orientado a objetos, imperativa y funcional.
