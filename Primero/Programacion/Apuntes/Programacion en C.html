Introducción

Introducción a C#

C# (pronunciado "C Sharp") es un lenguaje de programación creado por Microsoft que es utilizado en una amplia variedad de aplicaciones. Si nunca has programado antes, imagina que un lenguaje de programación es una forma de darle instrucciones a una computadora para que realice tareas. Así como nosotros usamos el español o el inglés para comunicarnos, los programadores usan lenguajes como C# para "hablar" con las computadoras.

¿Para qué se utiliza C#?

C# se usa en muchos tipos de aplicaciones diferentes, como por ejemplo:

  • Desarrollo de aplicaciones de escritorio: Si has usado programas en Windows, muchos de ellos probablemente fueron creados con C#.
  • Desarrollo de juegos: C# es muy popular en el desarrollo de videojuegos, sobre todo gracias a su uso en el motor de juegos Unity, uno de los más conocidos en la industria.
  • Aplicaciones web: Con C# se pueden crear sitios web y servicios en línea que millones de personas usan a diario.
  • Aplicaciones móviles: También es posible desarrollar aplicaciones móviles para Android y iOS usando C# con herramientas como Xamarin.
  • Inteligencia artificial, automatización y más: Gracias a su versatilidad, C# se utiliza en proyectos más avanzados como inteligencia artificial, automatización de procesos y sistemas embebidos (como los que controlan dispositivos electrónicos). Aunque es más probable encontrarte estos diseños en el lenguaje "Python"
Declaración de Variables

DECLARACIÓN Y ASIGNACIÓN DE VARIABLES

Antes de nada. ¿Qué son las variables?

Puedes pensar que las variables son como cajas en las que metemos dentro información.

Las variables se declaran usando una palabra clave que define el tipo de dato, el nombre que queremos darle, y finalizar la sentencia con un punto y coma (;).

Los tipos de datos que se utilizan son:

  • int
  • double
  • bool
  • strings

INT

El tipo de datos "INT" contiene números enteros.

Se declara:

int a;

La declaración se haría utilizando el operador "=", el cual no es un signo "igual" como en matemáticas.

Este signo "=" pertenece al operador "asignación", el cual guarda el contenido a la derecha del "=" en la variable indicada a su izquierda

Ejemplo:

a = 7

En este caso, a la variable "a" se le asigna el número 7

El resultado declaración + asignación sería el siguiente

int a;
a = 7;

Otra opción para ahorrar una línea de código sería:

int a = 7;

De ahora en adelante, se declarará y asignará todo en una misma línea

DOUBLE

El tipo de datos "DOUBLE" contiene números con sus decimales, también llamados números reales.

Ejemplo:

double a = 7.6;

Es importante expresar los números decimales con punto (.) y no con comas (,)

BOOL

El tipo de datos "BOOL" sólo puede contener dos valores. "True" y "False".

Se puede interpretar como pares "1/0", "verdadero/falso", "si/no", "encendido/apagado"...

Ejemplo:

bool a = true;
bool b = false;

STRINGS

El tipo de datos "STRING" contiene cadenas de textos.

Para especificar la cadena de texto, esta se debe declarar entre comillas ("")

Ejemplo:

string saludo = "Hola"
Console.WriteLine()

CONSOLE.WRITELINE

Console.WriteLine("Hello World")

Esta instrucción imprime en la consola de comandos el contenido que se encuentra dentro de su clase. Puede imprimir cualquier tipo de dato, tanto de un tipo como de varios.

Como ejemplo, si declaramos una variable como un número, podemos introducir esta variable dentro de la expresión.

int numero = 7;
Console.WriteLine(numero);

Esto dará como resultado una impresión en consola del valor de "numero", en este caso, 7.

Imprimir por consola varias variables

Dependiendo del caso, podríamos querer imprimir varias cosas por consolsa.

Una solución es hacer tantos "Console.WriteLine()" como variables tengamos, lo cual es poco eficiente

Para hacerlo todo en una misma línea, aquí tienes unos ejemplos.

// Ejemplo 1: Imprimir varias variables en una misma línea int edad = 25; string nombre = "Carlos"; Console.WriteLine("Nombre: " + nombre + ", Edad: " + edad);
// Ejemplo 2: Usando interpolación de cadenas (más eficiente y legible) int numero = 42; double precio = 19.99; Console.WriteLine($"El número es {numero} y el precio es {precio}");
// Ejemplo 3: Imprimir variables de diferentes tipos en una sola línea char inicial = 'A'; bool esEstudiante = true; Console.WriteLine("Inicial: " + inicial + ", ¿Es estudiante? " + esEstudiante);
Estándares de Nombres y Buenas Prácticas en C#

Estándares de Nombres y Buenas Prácticas en C#

En programación, es importante seguir buenas prácticas para nombrar variables, funciones y otros elementos, ya que esto facilita la lectura y mantenimiento del código. C# tiene algunos estándares recomendados que aseguran que el código sea claro y consistente. Aquí explicamos algunos de los estilos de nombrado más comunes y cómo se usan en este lenguaje.

  1. PascalCase:
    Este estilo comienza cada palabra con una letra mayúscula, sin espacios ni guiones. Es el estilo recomendado para nombrar clases, métodos (funciones) y Constructores y propiedades en C#. ("Constructores" y "Propiedades" se verá en un futuro tema) class MyClassName { public void CalculateTotal() { // Método en PascalCase } }
  2. camelCase
    En este estilo, la primera letra de la primera palabra es minúscula, y las siguientes palabras empiezan con mayúscula. Se utiliza comúnmente para nombrar variables y parámetros en C#. int precioFinal = 100; string nombreCliente = "Juan";
  3. snake_case
    Este estilo separa las palabras con guiones bajos y usa solo letras minúsculas. Aunque no es comunmente utilizado en C#, puede encontrarse en ciertos contextos o cuando se interopera con otros lenguajes, como bases de datos. int total_amount = 100; // RECUERDA: No es la práctica recomendada en C#
  4. kebab-case:
    Este estilo utiliza guiones en lugar de espacios para separar palabras. No se usa en C# ya que los guiones no son válidos en los nombres de variables o funciones. int precio-final = 100; // RECUERDA: No es la práctica recomendada en C#

Buenas prácticas en Nombres en C#

  1. Nombres claros y descriptivos:
    Siempre es mejor usar nombres que describan claramente lo que representa una variable, función o clase. ✔️ int userAge; ❌ int x;
  2. Usar PascalCase para clases y métodos:
    Las clases y los métodos deben seguir el estilo PascalCase para una mejor legibilidad. ✔️ public class CustomerData { } ❌ public class customerdata { }
  3. Usar camelCase para variables y parámetros:
    Las variables y los parámetros de las funciones deben comenzar con minúscula y seguir camelCase. ✔️ int productPrice; ❌ int ProductPrice;
  4. Evitar abreviaciones:
    Es mejor evitar usar abreviaciones poco claras, ya que pueden hacer el código difícil de entender. ✔️ int orderQuantity; ❌ int qty;
  5. Prefijo 'I' para interfaces:
    Las interfaces deben comenzar con la letra I seguida del nombre en PascalCase. ✔️ public interface ICustomerRepository ❌ public interface CustomerRepository
Funciones

FUNCIONES

En programación, las funciones (también llamadas métodos en C#) son bloques de código que realizan una tarea específica y pueden ser reutilizados en diferentes partes del programa. Las funciones permiten organizar el código, evitar repeticiones y hacerlo más legible.

¿Cómo se crea una función en C#?

Es importante conocer muy bien la sintaxis de las funciones.
Memorizando este orden, seréis capaces de crear las estructuras de las funciones sin ningún problema ni errores.

  1. Palabras clave necesarias: Por ahora, usaremos las palabras public static antes de la definición de la función, aunque su significado lo veremos más adelante. De momento, solo necesitamos saber que son obligatorias para que la función funcione. public static
  2. Tipo de retorno: Indica el tipo de dato que va a devolver la función. Si la función no devuelve ningún valor, utilizamos la palabra clave void. public static bool public static double
  3. Nombre de la función: El nombre que le damos a la función, y debe ser representativo de lo que hace (por ejemplo, SumaNumeros).
    Recordemos que hay que cumplir los estándares del lenguaje de programación C# vistos en el punto anterior. public static double SumaNumeros
  4. Parámetros: Son las "entradas" o "argumentos" que le damos a la función. Si la función necesita datos para realizar su tarea, los especificamos entre paréntesis. Puedes introducir tantos argumentos como la función requiera para su uso, siempre declarando antes su tipo de dato y nombre de variable, y siempre separandolos con una coma. public static double SumaNumeros(int a, int b) public static int SumaNumeros(double a, double b, double c)
  5. Cuerpo de la función: Es el bloque de código entre llaves { } que contiene las instrucciones que ejecutará la función. public static double SumaNumeros(int a, int b) { int suma = a + b; }
  6. Fin de la función: para darle fin a una función se utilizará la palabra reservada "return". Una vez que la función se encuentra esta palabra, la función devolverá el resultado y esta dejará de ejecutarse. public static double SumaNumeros(double a, double b) { int suma = a + b; return suma; } Si queremos simplificar el código, podríamos hacerlo de la siguiente manera: public static double SumaNumeros(double a, double b) { return a + b; }

**IMPORTANTE**

Recordad que la función siempre ha de devolver el tipo de dato que se ha asignado. Si declaraste que devolvería un "bool", pero el return contiene un "int", la función dará un error y no funcionará.

Operadores

En C#, los operadores son símbolos que le dicen al compilador qué tipo de operación realizar. Estos operadores trabajan con variables y valores para realizar cálculos matemáticos, comparaciones, y tomar decisiones lógicas, entre otras cosas.

Tipos de Operadores

  • Aritméticos
  • Comparación
  • Lógicos
  • Asignación
  • Unarios

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas como la suma, resta, multiplicación y división.

  1. Operador "==": Igual a.
    Este operador se encarga de comprobar que los elementos colocados a sus lados sean del mismo valor.
    int a = 10; int b = 20; int c = 10; bool result1 = a == b; // return false bool result = a == c; // return true
  2. Operador "!=": Distinto a.
    Este operador se encarga de comparar que los elementos colocados a sus lados sean diferentes.
    int a = 10; int b = 20; int c = 10; bool result1 = a != b; // return True bool result2 = a !=c; // return false
  3. Operador ">": Mayor que.
    Este operador se encarga de comprobar si el valor colocado a su izquierda es mayor que el colocado a su derecha. int a = 10; int b = 20; bool result1 = a > b; // return False bool result2 = b > a; // return True
  4. Operador "<": Menor que.
    Este operador se encarga de comprobar si el valor colocado a su izquierda es menor que el colocado a su derecha int a = 10; int b = 20; bool result1 = a < b; // return True bool result2 = b < a; // return false
  5. Operador ">=": Mayor o igual
    Este operador se encarga de comprobar si el valor colocado su izquierda es mayor o igual al colocado a su derecha.
    Es importante entender que es "mayor o igual", y no "igual o mayor". int a = 10; int b = 20; bool result1 = a >= b; // return False bool result2 = b >= a; // return True
  6. Operador "<=": Menor o igual
    Este operador se encarga de comprobar si el valor colocado a su izquierda es menor o igual al colocado en su derecha
    Es importante entender que es "menor o igual", y no "igual o menor".
    c int a = 10; int b = 20; bool result1 = a <= b; // return True bool result2 = b <= a; // return False

Operadores

Se utilizan para combinar condiciones lógicas y devuelven un valor booleano (true o false). Los operadores más comunes son:

  1. &&
    El operador lógico &&, en inglés "and", Evalúa si ambas condiciones son true. Si todas las condiciones son verdaderas, el resultado será true. Si al menos una condición es false, el resultado será false.
    true && true = True true && false = false false && true = false false && false = false Podemos usarlo con un ejemplo práctico con un caso real. int edad = 25; bool tieneLicencia = true; if (edad >= 18 && tieneLicencia) { Console.WriteLine("Puedes conducir."); } else { Console.WriteLine("No puedes conducir."); } Puedes cambiar las variables edad y tieneLicencia para comprobar todos los casos.
    Os dejo otro ejemplo: bool estaLloviendo = true; bool tengoParaguas = false; if (estaLloviendo && tengoParaguas) { Console.WriteLine("Salgo a caminar."); } else { Console.WriteLine("No puedo salir, está lloviendo."); }
  2. ||
    El operador || (En inglés, "or"), Evalúa si al menos una de las condiciones es true. Si cualquiera de las dos (o ambas) condiciones es true, el resultado será true. Sólo será false si todas las condiciones son false.
    true || true = true true || false = true false || true = true false || false = false bool estaLloviendo = true; bool tengoParaguas = false; if (estaLloviendo || tengoParaguas) { Console.WriteLine("Salgo a caminar."); } else { Console.WriteLine("No salgo."); } Aunque tengoParaguas es false, la condición estaLloviendo es true. Como al menos una condición es true, el bloque dentro del if se ejecuta y se imprime "Salgo a caminar.".

Combinación de ||y &&

Es posible combinar múltiples operadores lógicos en una sola expresión.
Cuando esto ocurre, es importante recordar la precedencia de los operadores. El operador && (AND) tiene mayor precedencia que el operador || (OR). Esto significa que primero se evalúan las expresiones con &&, y luego las con ||, a menos que se utilicen paréntesis para alterar el orden.

int edad = 17; bool tienePermisoPadres = true; bool esVIP = false; if (edad >= 18 || (tienePermisoPadres && esVIP)) { Console.WriteLine("Puedes acceder al evento."); } else { Console.WriteLine("No puedes acceder al evento."); }

Explicación:
Primero se evalúa la expresión dentro de los paréntesis: tienePermisoPadres && esMiembroVIP. tienePermisoPadres es true, pero esMiembroVIP es false. Por lo tanto, la combinación tienePermisoPadres && esMiembroVIP es false.
Luego, se evalúa la expresión completa: edad >= 18 || false. edad >= 18 es false, y false || false da como resultado false.
Como el resultado final es false, se ejecuta el bloque else.

Operadores de Asignación

Estos operadores se utilizan para asignar valores a las variables. Algunos combinan la asignación con una operación matemática.

  1. Operador =
    Es una asignación simple. int a = 10;
  2. Operador +=
    Este operador se encarga de, a la vez de sumar la cantidad indicada a la derecha de la expresión, declarar el resultado de la operación en la variable. int x = 10; x += 5; // x = x + 5 = 15 x += 5; // ahora, x = x + 5 = 20
  3. Operador -=
    Al igual que su contraparte, este se encarga de restar y asignar el resultado en la variable
    int x = 10; x -= 5; // x = x - 5 = 5 x -= 5; // ahora, x = x - 5 = 0
  4. Operador ++
    Este operador se encarga de incrementar el valor de la variable en uno. Generalmente, esta operación se especifica después de la variable, como veremos en el ejemplo de abajo.
    int a = 7; a++ Console.WriteLine(a); // a = 8 También se puede utilizar a la inversa
    int a = 7; ++a; Console.WriteLine(a); // a = 8 Aunque en términos generales no hay diferencia, algunos casos pueden dar resultados diferentes.
  5. !
    El operador ! significa "not". Invierte el valor de la variable. bool a = true; Console.WriteLine(!a); // devuelve false
  6. -
    No hay que confundirlo con la resta. Este operador va pegado, sin espacios, a la variable que queremos afectar.
