Introducción

Introducción a C#

C# (pronunciado "C Sharp") es un lenguaje de programación creado por Microsoft que es utilizado en una amplia variedad de aplicaciones. Si nunca has programado antes, imagina que un lenguaje de programación es una forma de darle instrucciones a una computadora para que realice tareas. Así como nosotros usamos el español o el inglés para comunicarnos, los programadores usan lenguajes como C# para "hablar" con las computadoras.

¿Para qué se utiliza C#?

C# se usa en muchos tipos de aplicaciones diferentes, como por ejemplo:

  • Desarrollo de aplicaciones de escritorio: Si has usado programas en Windows, muchos de ellos probablemente fueron creados con C#.
  • Desarrollo de juegos: C# es muy popular en el desarrollo de videojuegos, sobre todo gracias a su uso en el motor de juegos Unity, uno de los más conocidos en la industria.
  • Aplicaciones web: Con C# se pueden crear sitios web y servicios en línea que millones de personas usan a diario.
  • Aplicaciones móviles: También es posible desarrollar aplicaciones móviles para Android y iOS usando C# con herramientas como Xamarin.
  • Inteligencia artificial, automatización y más: Gracias a su versatilidad, C# se utiliza en proyectos más avanzados como inteligencia artificial, automatización de procesos y sistemas embebidos (como los que controlan dispositivos electrónicos). Aunque es más probable encontrarte estos diseños en el lenguaje "Python"
Declaración de Variables

DECLARACIÓN Y ASIGNACIÓN DE VARIABLES

Antes de nada. ¿Qué son las variables?

Puedes pensar que las variables son como cajas en las que metemos dentro información.

Las variables se declaran usando una palabra clave que define el tipo de dato, el nombre que queremos darle, y finalizar la sentencia con un punto y coma (;).

Los tipos de datos que se utilizan son:

  • int
  • double
  • bool
  • strings

INT

El tipo de datos "INT" contiene números enteros.

Se declara:

int a;

La declaración se haría utilizando el operador "=", el cual no es un signo "igual" como en matemáticas.

Este signo "=" pertenece al operador "asignación", el cual guarda el contenido a la derecha del "=" en la variable indicada a su izquierda

Ejemplo:

a = 7

En este caso, a la variable "a" se le asigna el número 7

El resultado declaración + asignación sería el siguiente

int a;
a = 7;

Otra opción para ahorrar una línea de código sería:

int a = 7;

De ahora en adelante, se declarará y asignará todo en una misma línea

DOUBLE

El tipo de datos "DOUBLE" contiene números con sus decimales, también llamados números reales.

Ejemplo:

double a = 7.6;

Es importante expresar los números decimales con punto (.) y no con comas (,)

BOOL

El tipo de datos "BOOL" sólo puede contener dos valores. "True" y "False".

Se puede interpretar como pares "1/0", "verdadero/falso", "si/no"

Ejemplo:

bool a = true;
bool b = false;

STRINGS

El tipo de datos "STRING" contiene cadenas de textos.

Para especificar la cadena de texto, esta se debe declarar entre comillas ("")

Ejemplo:

string saludo = "Hola"
Console.WriteLine()

CONSOLE.WRITELINE

Console.WriteLine("Hello World")

Esta instrucción imprime en la consola de comandos el contenido que se encuentra dentro de su clase. Puede imprimir cualquier tipo de dato, tanto de un tipo como de varios.

Como ejemplo, si declaramos una variable como un número, podemos introducir esta variable dentro de la expresión.

int numero = 7;
Console.WriteLine(numero);

Esto dará como resultado una impresión en consola del valor de "numero", en este caso, 7.

Imprimir por consola varias variables

Dependiendo del caso, podríamos querer imprimir varias cosas por consolsa.

Una solución es hacer tantos "Console.WriteLine()" como variables tengamos, lo cual es poco eficiente

Para hacerlo todo en una misma línea, aquí tienes unos ejemplos.

// Ejemplo 1: Imprimir varias variables en una misma línea int edad = 25; string nombre = "Carlos"; Console.WriteLine("Nombre: " + nombre + ", Edad: " + edad);
// Ejemplo 2: Usando interpolación de cadenas (más eficiente y legible) int numero = 42; double precio = 19.99; Console.WriteLine($"El número es {numero} y el precio es {precio}");
// Ejemplo 3: Imprimir variables de diferentes tipos en una sola línea char inicial = 'A'; bool esEstudiante = true; Console.WriteLine("Inicial: " + inicial + ", ¿Es estudiante? " + esEstudiante);
Funciones

FUNCIONES

En programación, las funciones (también llamadas métodos en C#) son bloques de código que realizan una tarea específica y pueden ser reutilizados en diferentes partes del programa. Las funciones permiten organizar el código, evitar repeticiones y hacerlo más legible.

¿Cómo se crea una función en C#?

Es importante conocer muy bien la sintaxis de las funciones.
Memorizando este orden, seréis capaces de crear las estructuras de las funciones sin ningún problema ni errores.

  1. Palabras clave necesarias: Por ahora, usaremos las palabras public static antes de la definición de la función, aunque su significado lo veremos más adelante. De momento, solo necesitamos saber que son obligatorias para que la función funcione. public static
  2. Tipo de retorno: Indica el tipo de dato que va a devolver la función. Si la función no devuelve ningún valor, utilizamos la palabra clave void. public static bool public static double
  3. Nombre de la función: El nombre que le damos a la función, y debe ser representativo de lo que hace (por ejemplo, SumaNumeros).
    Recordemos que hay que cumplir los estándares del lenguaje de programación C# vistos en el punto anterior. public static double SumaNumeros
  4. Parámetros: Son las "entradas" o "argumentos" que le damos a la función. Si la función necesita datos para realizar su tarea, los especificamos entre paréntesis. Puedes introducir tantos argumentos como la función requiera para su uso, siempre declarando antes su tipo de dato y nombre de variable, y siempre separandolos con una coma. public static double SumaNumeros(int a, int b) public static int SumaNumeros(double a, double b, double c)
  5. Cuerpo de la función: Es el bloque de código entre llaves { } que contiene las instrucciones que ejecutará la función. public static double SumaNumeros(int a, int b) { int suma = a + b; }
  6. Fin de la función: para darle fin a una función se utilizará la palabra reservada "return". Una vez que la función se encuentra esta palabra, la función devolverá el resultado y esta dejará de ejecutarse. public static double SumaNumeros(double a, double b) { int suma = a + b; return suma; } Si queremos simplificar el código, podríamos hacerlo de la siguiente manera: public static double SumaNumeros(double a, double b) { return a + b; }

**IMPORTANTE**

Recordad que la función siempre ha de devolver el tipo de dato que se ha asignado. Si declaraste que devolvería un "bool", pero el return contiene un "int", la función dará un error y no funcionará.
