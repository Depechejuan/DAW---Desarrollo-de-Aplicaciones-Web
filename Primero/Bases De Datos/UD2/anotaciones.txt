¿Qué hacemos cuando queremos hacer una base de datos de un cliente?

1.- Análisis de requisitos:
    Que nos expliquen en profundidad su negocio e idea
2.- Diagrama E-R:
    Es un bucle, una vez listo hay que volver a analizarla.
    Usamos erdplus para crear el modelo E-R
3.- Modelo Relacional:
    Transformar el E-R a un formato lo más cercano a tablas
    Si hay un error, se tiene que "normalizar" (normalización, arreglarlo).
4.- DDL - Creación de tablas
    Se necesita tomar una decisión sobre qué sistema de tablas vamos a usar (SQL server, Oracle, MariaDB....)
    Si empiezo con uno, y queremos pasar a otro (nos sobra o nos falta), hay que hacer una migración



Modelo de datos = las tablas
